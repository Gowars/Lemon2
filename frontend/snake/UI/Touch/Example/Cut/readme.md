# 图片剪切

-   使用矩阵对图片进行缩放
-   缩放以图片的左上角为中心
-   指定图片的位移基准，可以是屏幕左上角，也可以是一个指定的元素的左上角
-   在这里我们指定为剪切选择框的左上角
-   获取选择框相对于图片的坐标信息 {x, y, width, height, imgWidth, imgHeight, scale: 最终图片大小与选择框的比例}

-   因为 canvas 的同源限制，通过 XMLHttpRequest 把 url -> file
-   图片 exif 信息读取，对图片进行旋转矫正
-   通过 canvas 对图片进行剪切
